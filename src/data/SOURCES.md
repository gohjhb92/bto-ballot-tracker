# Data Sources

All subscription rate data is compiled from publicly available sources for reference purposes.

## Primary Sources

| Exercise | Source |
|---|---|
| 2020–2022 exercises | [HDB official BTO results releases](https://www.hdb.gov.sg/cs/infoweb/residential/buying-a-flat/finding-a-flat/buying-a-new-flat/bto-launches) |
| 2023 exercises | HDB official releases + Seedly community reporting |
| Feb 2024 | HDB official results (Feb 2024 BTO release) |
| Jun 2024 | HDB official results (Jun 2024 BTO release); Ohmyhome analysis |
| Oct 2024 | HDB official results (first S/P/P exercise); PropertyGuru analysis |
| Feb 2025 | HDB official results (Feb 2025 BTO release) |

## Secondary Sources

- **Seedly** (seedly.sg) — community-aggregated subscription rate summaries
- **PropertyGuru** (propertyguru.com.sg) — post-exercise analysis articles
- **Ohmyhome** (ohmyhome.com) — BTO subscription rate breakdowns
- **99.co** — BTO launch coverage with rates

## Notes

- Rates are expressed as **subscription rate** = number of valid applications ÷ number of available units.
- Where town-level data is available, it reflects the specific project's rates; summary data is the exercise-wide average for that estate type × flat type combination.
- Pre-Oct 2024 data uses the **mature / non-mature** classification. Post-Oct 2024 data uses **Standard / Plus / Prime**.
- Some exercises show partial town-level data only (where comprehensive project-level data was publicly disclosed by HDB).
- The 1.7× threshold is from HDB's own published guidance on competitive ballots.

## Disclaimer

This data is compiled for informational purposes. Always verify with [HDB's official website](https://www.hdb.gov.sg) before making housing decisions.
